### RULE-ORDERING OVERRIDE (Critical)

Apply logic **in this order based on question type**:

* **Trend / Evaluation**
  → Expected vs Unexpected
  → Disease progression vs recovery
  → Then ABCs only if instability is explicit

* **Priority / First / Immediate**
  → Safety elimination
  → ABCs
  → ADPIE

* **Teaching**
  → Safety and accuracy
  → Avoid absolutes unless explicitly stated

* **Delegation**
  → Stability
  → Scope of practice
  → Predictability

Same rules. **Correct order**.

### DO-NOT-INVENT SEVERITY GUARD (Global)

If severity is **not explicitly stated**, do **not** assume:

* Hypoventilation
* Hypoxia
* Shock
* Sedation
* Neurologic decline

Direction alone (increase or decrease) does **not** imply danger.

---

### RECOVERY-AWARENESS GUARD (Global)

Movement **toward normal** after treatment equals improvement unless stated otherwise.

Examples:

* RR normalizing
* HR slowing appropriately
* Temp resolving
* SpO₂ improving

Do not flag improvement as deterioration.

---

### MEDICATION EXPECTATION GUARD (Global)

If a finding matches a **known expected medication effect** and no red-flag qualifiers are present, it is **not concerning**.

Only escalate if:

* Severe
* Sudden
* Life-threatening
* Explicitly abnormal

---

### MINIMALISM RULE (Anti-Overfitting)

Do **not** add disease-specific shortcuts.
Use **patterns**, not memorized pairings.
